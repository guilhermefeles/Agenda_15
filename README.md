# Verdant — Tecnologia que respeita o planeta

Site institucional da **Verdant**, uma consultoria de tecnologia sustentável que ajuda empresas a reduzir seu impacto ambiental sem abrir mão da inovação.

## Visão Geral

Página única (one-page) com navegação lateral, apresentando a empresa, seus parceiros e perguntas frequentes.

## Estrutura

```
site_Verdant/
├── index.html
└── imagens/
    └── verdant-forest.webp
```

## Seções

- **Home** — Hero com imagem de floresta e overlay escurecido
- **Quem Somos / Missão / Valores** — Apresentação da empresa com cards de valores (Sustentabilidade, Inovação, Compromisso)
- **Parceiros** — Cards com EcoTech, SolarBR e ReciclaMax
- **FAQ** — Perguntas frequentes em lista interativa
- **Rodapé** — Links de redes sociais, e-mail e telefone

## Tecnologias

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura da página |
| W3.CSS | Framework de estilos |
| Font Awesome 4.7 | Ícones |
| Google Fonts (Montserrat) | Tipografia |

## Como abrir

Basta abrir o arquivo `index.html` diretamente no navegador — não requer servidor ou dependências instaladas.

## Design

- Tema escuro com paleta verde floresta (`#1b4332`, `#2c6d52`)
- Sidebar fixa de navegação lateral 
- Background com parallax fixo na seção hero
- Layout responsivo via W3.CSS grid
